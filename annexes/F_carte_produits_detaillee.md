# Annexe F : Carte Produits Définitive Espressolab Tunisie

## Introduction

Cette annexe présente la **carte produits** d'Espressolab Tunisie, optimisée pour le marché tunisien avec un **ticket moyen de 20,00 TND** (5,86 EUR). L'offre combine des **prix d'entrée accessibles** pour les produits de base et une **valorisation des spécialités** pour maximiser l'équilibre entre accessibilité et rentabilité.

## Méthodologie Tarifaire Définitive

- **Objectif** : Ticket moyen 20,00 TND (5,86 EUR) avec stratégie de prix différenciée
- **Marge variable** : 60% sur produits de base, 65-70% sur spécialités
- **Coût matières** : 30-40% du prix de vente selon les catégories
- **Café franchiseur** : 126,86 TND/kg (23 USD + taxes)
- **Positionnement** : Qualité Third Wave avec prix d'entrée accessibles
- **Clientèle cible** : 280 clients/jour (équilibre optimal volume/marge)
- **Taux de change** : 1 EUR = 3,41 TND (taux utilisé pour la conversion)
